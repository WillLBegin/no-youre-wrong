# No, You're Wrong!

A fun debate website where you get matched against a random opponent and argue over a randomly assigned topic.

## How It Works

1. **Get matched** — You're paired with a random opponent in real time.
2. **Receive a topic** — Both players get the same random subject to argue about.
3. **Argue it out** — Make your case and try to come out on top.
4. **Winner decided** — TBD!
